# AI Search Optimization Skill for Claude Code

Get cited by ChatGPT, Perplexity, Claude, Gemini, and AI assistants. 8 disciplines for AI search visibility, citation optimization, and agentic commerce readiness.

## What's Included

- **8 AI Search Disciplines:** GEO, LLMO, AIVO, CSO, AAIO, AI Shopping, Reddit Optimization, Multimodal Search
- **10 Reference Files:** Deep-dive guides with citation tactics, brand monitoring, and AI commerce setup
- **Business Type Prioritization:** Which AI disciplines matter most for SaaS, E-commerce, Local Services, Healthcare, Professional Services, Media
- **Citation Optimization Checklist:** Actionable steps for every page
- **Key Statistics:** Data-backed insights with sources throughout

## Installation

1. Download `SKILL.md` and the `references` folder
2. Drag and drop into Claude (Desktop app, VS Code, Cursor, or Terminal)
3. Tell Claude: "Install this skill"
4. Invoke with `/ai-search-optimization-skill`

## Usage

```
/ai-search-optimization-skill
```

## The 8 Disciplines

### Core AI Optimization
1. **GEO** - Generative Engine Optimization: citation hooks, original data, author authority
2. **LLMO** - Large Language Model Optimization: training data, RAG retrieval, semantic clarity
3. **AIVO** - AI Visibility Optimization: brand accuracy monitoring, hallucination detection
4. **CSO** - Conversational Search Optimization: multi-turn dialogue, follow-up intent

### AI Commerce and Agents
5. **AAIO** - Agentic AI Optimization: AI agents that browse, book, buy
6. **AI Shopping** - ChatGPT Checkout, Google AI Shopping, Perplexity shopping

### AI Source Channels
7. **Reddit Optimization** - #1 AI citation source, subreddit authority, authentic participation
8. **Multimodal Search** - Cross-modal content: photo + text, voice + image, video analysis
